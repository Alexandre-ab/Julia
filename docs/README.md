## 📝 Convention

**Format des documents** :
- Markdown (`.md`) pour la lisibilité
- Titres hiérarchiques (`#`, `##`, `###`)
- Emojis pour la navigation visuelle
- Code blocks avec coloration syntaxique
- Tables pour les données structurées

**Nommage des fichiers** :
- **MAJUSCULES** pour les documents principaux (ex: `README.md`, `INDEX.md`)
- **Kebab-case** pour les fichiers spécifiques (ex: `test-connection.md`)
- **Extension** : toujours `.md` (ou `.txt` pour les formats ASCII)
